# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-30

### Added
- Initial release of effect-jsonapi
- Core JSON:API schema definitions using Effect Schema
- Builder utilities for constructing JSON:API documents
- HTTP integration with @effect/platform
- Query parameter parsing (filter, sort, pagination, include, fields)
- Serialization utilities for converting data to JSON:API format
- Comprehensive test suite with vitest
- Example blog API demonstrating library usage
- Full TypeScript support with type definitions
- Complete documentation in README

### Features
- ✅ JSON:API v1.1 specification compliance
- ✅ Type-safe schema definitions
- ✅ Effect-based builders for composable API construction
- ✅ HTTP request/response handling
- ✅ Resource serialization
- ✅ Relationship handling (to-one, to-many)
- ✅ Error formatting
- ✅ Query parameter parsing and validation
